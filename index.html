ABOUT FAWNOOS


Support and consulting services for open-source identity and access management.

Supported echnologies and platforms include Apereo CAS, Pac4j, Shibboleth, etc. Primary focus on web single sign-on, multifactor authentication, federated identity, authentication protocols and standards, application integration, and more.

Fawnoos is a term of Greek origin meaning 'light' or 'lantern' and is a symbol for hope and guidance in Persian classical literature.
Apereo CAS

Apereo CAS is an open-source multilingual enterprise-ready solution for Web SSO and access management.


Services include defect analysis, training and workshops, feature development, Q/A support, application integration, UI design and branding, performance evaluation and load testing, deployment assessments, etc.

Pac4j

Pac4j is Java-based security library for authentication and profile management with support for many frameworks and platforms and heavily used in Apereo CAS for social authentication.


Services include defect analysis, feature development, Q/A support, application integration, bug fixes, etc.

Shibboleth

Shibboleth Identity Provider provides federated Web SSO capabilities with support for authentication protocols such as SAML2, CAS, etc.


Services include feature and extension development, training, Q/A support, application integration with service providers, etc.

General Consulting

Consulting services for identity and access management assessments, training and general development activities.


Primarily focused on open-source, Java ecosystem and languages and technologies such as Spring Framework, CI/CD pipelines, Docker, etc.
